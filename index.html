Architecture

Six stages,
fully automated

Each pipeline run processes the last 48 hours of LinkedIn job alert emails, scrapes the full listings, scores them against your preferences with Claude, and generates personalised application documents ‚Äî only for jobs it hasn't seen before.

Your existing application statuses and notes are always preserved. Jobs with no action after 7 days are automatically purged.

Quick Start

Up and running
in minutes

# Clone and install
git clone https://github.com/yourusername/hr-assistant.git
cd hr-assistant
pip install anthropic pyyaml google-auth-oauthlib google-api-python-client playwright python-docx
playwright install chromium

# Set your API key
set ANTHROPIC_API_KEY=sk-ant-your-key-here

# Run the full pipeline
python run_pipeline.py

# Or skip to scoring if you already have scraped data
python run_pipeline.py --from-filter

# Set up 48-hour automation (run as Administrator)
setup_scheduler.bat
